Trabalhando com listas

Listas ordenadas

Para trabalhar com listas ordenadas primeiro criamos a tag < ol >. Que significa order list (lista ordenada) Depois dentro de < ol > utilizamos a tag < li > que significa item da lista.

  1. Acordar
  2. Ligar para joao
  3. Escovar os dentes
  4. abraçar minha mãe

Podemos formatar as nossas listas por meio de tags. Basta dentro de < ol > utilizar o parametro type e escolher o tipo de item da lista voce deseja.

Listas não ordenadas

Para criarmos listas não ordenadas utilizamos a tag < ul > e também iremos utilizar novamente a tag < li > para listar os itens da lista.

  • estudar java
  • estudar html
  • estudar banco de dados
  • estudar IHC

Também podemos personalizar nossas listas não ordenadas. Para tal utilizamos dentro da tag < ul > novamente o parâmetro type e poderemos escolher o estilo de formataçã por meio das palavras (circle, square, disk(padrão)).

type = "circle"
