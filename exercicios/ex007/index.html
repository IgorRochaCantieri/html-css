Principais formatações

Negrito ou destaque

Nessa frase temos um texto em negrito usando a tag < b > (Não semântica)

Nesta frase temos um texto em negrito usando a tag < STRONG > (Semântica)

Italico ou Ênfase

Nesta frase temos um termo em Italico usando a tag < i > (nao semantica)

Nesta frase temos mais uma palavra em italico usando a tag < en > (Semantico)

Alterando textos de maneira correta

Juvenal era um sujeito de muita sorte. E ja começou de pequeno onde morou na

Rua Marques de Lira Filho

um local de fácil acesso ao centro da cidade

Como quis dar mais atenção ao local onde Juvenal nasceu, utilizzei a teg < address > para tal feito.

Alterando o mesmo texto com tag diferente

Juvenal era um sujeito de muita sorte. E ja começou de pequeno onde morou na Rua Marques de Lira Filho um local de fácil acesso ao centro da cidade

No texto acima utilizei a tag < strong > para deixar em negrito

A tag < em > deixa o texto em Italico

texto marcado

Podemos também criar um testo marcado usando a tag < MARK > depois dentro da tag < MARK > utilizei a tag < STYLE > para alterar a cor da fonte do texto marcado.

Texto grande e pequeno

Este texto tera uma pate pequena para tal utilizei a tag < SMALL > nao colocarei a grande pois faz parte de css

texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado, para tal utilizei a tag < DEL >

texto inserido

Podemos marcarum texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, para tal utilizei a tag < INS >

texto sobrescrito

Para escrever coisas do tipo x2 + 3 (x ao quadrado mais três) utilizei a tag < SUP > somente no numero

texto subscrito

Para inserir coisa do tipo h2O. Utilizei a tag < SUB > somente no numero
