Trabalhando com links externos

Voce pode acessar um repositório publico no GitHub, iremos acessar o repositório publico do professor Guanabara aqui

Iremos acessar o canal gratuito no YouTube do prf Guanabara clicando aqui

Utilizamos a tag < a > (a = ancora) para criar uma ancora e colocar o site desejado.
Dentro desta tag existirá um parâmetro obrigatório: href (referencia hiper texto), nele iremos digitar o site que iremos acessar. href = "link do site".
Para ajudar na navegação também utilizaremos mais dois parâmetros: target nele colocaremos a palavra _blank que permitira que o próximo site não feche o nosso criando assim mais uma aba. target = "_blank".
O outro parâmetro é o rel, nele iremos colocar a palavra externals ela ajudara a referenciar o navegador informando que ele está saindo para fora do nosso site. rel = "externals".
